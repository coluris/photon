# Setup

```
npm install
make install-deps
```

You're good to go!

# Compiles and hot-reloads for development

```
npm run electron:serve
```

# Compiles and minifies

```
npm run electron:build
```

# Lints and fixes files

```
npm run lint
```

# Runs Storybook to view UI components

```
npm run storybook
```
